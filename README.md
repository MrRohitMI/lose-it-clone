
# Project Description

This is a clone of weight loss website called Lose It.

## Features

- Chakra UI for layout
- Carousel in Home Page
- Login/Signup functionality
- Blogs page


## Deployment

To deploy this project run
```bash
  npm install
```

```bash
  npm run start
```
